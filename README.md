# Automatic thermostat

This semester project for the Embedded Systems course focuses on the design and implementation of thermostat firmware for the ATmega328P microcontroller. The goal is to build a fully functional standalone thermostat using the hardware modules from the provided kit, capable of measuring ambient temperature, controlling heating or cooling based on a configured setpoint, and providing a user interface for control and configuration. The implementation makes use of the libraries developed during the semester as part of the course assignments.
